QR Asistencia es una aplicación para registrar asistencia laboral mediante códigos QR. Esta sección pública reúne la política de privacidad, los términos del servicio y la información básica del tratamiento de datos utilizada para la integración con Google Cloud y Google Sheets API.

Propósito de la aplicación

La aplicación permite a empleados autorizados registrar entradas, salidas y eventos de asistencia asociados a un proyecto o centro de trabajo. La información capturada se utiliza únicamente con fines operativos y administrativos internos de control de asistencia.
