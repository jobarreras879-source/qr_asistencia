Página de Privacidad Específica - QR Asistencia

Versión Oficial para Google Cloud Verification

Esta página contiene la Política de Privacidad completa para la aplicación móvil QR Asistencia.

1. Identificación del Responsable

El responsable del tratamiento de los datos es AVS Ingeniería (jobarreras879@gmail.com).

Nuestra aplicación recopila únicamente la información necesaria para el control de asistencia:

  • Nombre de usuario y rol dentro del sistema.
  • Registros de entrada y salida (fecha, hora y proyecto asociado).

2. Uso de los datos de Google

La aplicación utiliza el acceso a Google Sheets exclusivamente para:

  • Escribir los registros de asistencia en la hoja de cálculo configurada por el administrador.
  • No compartimos, vendemos ni utilizamos estos datos para ningún otro propósito técnico o comercial.

3. Seguridad

Implementamos medidas de seguridad para proteger los datos, incluyendo el uso de Supabase para la gestión segura de base de datos y encriptación de contraseñas.

4. Contacto

Para dudas sobre esta política, contacte a: jobarreras879@gmail.com
