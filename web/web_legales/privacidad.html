Página de Privacidad Específica - QR Asistencia

Versión Oficial para Google Cloud Verification

Esta página contiene la Política de Privacidad completa para la aplicación móvil QR Asistencia.

1. Información que Recopilamos

La Aplicación recopila y utiliza los siguientes datos exclusivamente para el control de asistencia laboral:

  • Información de Perfil: Nombre de usuario y correo electrónico proporcionado durante el inicio de sesión.
  • Datos de Asistencia: Registros de entrada y salida, incluyendo fecha, hora y ubicación/proyecto asociado.

2. Uso de Datos de Google (Google Sheets API)

Nuestra aplicación solicita acceso a su cuenta de Google para interactuar con la API de Google Sheets. Este acceso se utiliza únicamente para:

  • Crear y actualizar registros de asistencia en una hoja de cálculo específica gestionada por su empresa (AVS Ingeniería).
  • No leemos ni accedemos a ningún otro archivo en su Google Drive que no sea la hoja de cálculo de asistencia configurada para la aplicación.

Cumplimiento de la Política de Datos de Usuario del Servicio de API de Google: El uso de la información recibida de las API de Google por parte de la Aplicación cumplirá con la Política de datos de usuario de los servicios de API de Google, incluidos los requisitos de uso limitado.

3. Intercambio y Transferencia de Datos

No vendemos, intercambiamos ni transferimos sus datos personales a terceros. Los datos de asistencia solo son accesibles para el administrador de AVS Ingeniería a través de Google Sheets y Supabase.

4. Retención y Eliminación de Datos

Los datos se conservan mientras sea necesario para los fines administrativos de la empresa. Los usuarios pueden solicitar la eliminación de su cuenta y datos asociados contactando al administrador.

5. Seguridad

Implementamos protocolos de seguridad estándar para proteger la integridad de la información, utilizando servicios de backend seguros y encriptación en tránsito.
